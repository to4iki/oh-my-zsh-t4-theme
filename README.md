## Installation

After installing [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh), you can install this by:

```bash
$ curl -fsSL https://raw.github.com/to4iki/oh-my-zsh-t4-theme/master/t4.zsh-theme >> ~/.oh-my-zsh/themes/t4.zsh-theme
```

And configure `.zshrc`:

```bash
ZSH_THEME="t4"
```
